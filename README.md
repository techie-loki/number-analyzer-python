# Number Analyzer CLI Tool (Python)

A simple command-line Python application that performs multiple operations on a number using a menu-driven interface.

---

## 🎯 Purpose

This project was built to practice core Python concepts like loops, functions, and conditional logic by creating a real-world command-line tool.

---

## 🚀 Features

* Reverse a number
* Check if a number is a palindrome
* Calculate sum of digits
* Count number of digits
* Full analysis (all operations at once)

---

## 🧠 Concepts Used

* Loops (`while`)
* Conditional statements (`if-elif`)
* Functions (modular programming)
* User input handling

---

## 📚 What I Learned

* How to structure a Python program using functions
* How to build a menu-driven CLI application
* How to reuse functions efficiently
* Basic problem-solving and debugging skills

---

## 🖥️ How to Run

1. Clone the repository:

```bash
git clone https://github.com/techie-loki/number-analyzer-python.git
```

2. Navigate to the project folder:

```bash
cd number-analyzer-python
```

3. Run the program:

```bash
python number_analyzer_v2.py
```

---

## 📌 Example

```
Number Analyzer
1. Analyze
2. Reverse
3. Palindrome
4. Sum
5. Count
6. Exit

Enter option: 1
Enter number: 121

Reverse number : 121
```
